# gpx2png

A tool that converts gpx to png. Comes with a discord bot.

## requirement
- diccord.py
- gpxpy
- geopy
- pillow
- dotenv
- emoji

## how to use
Method 1: Run bot.py and
  - send gpx file(s), or
  - reply "check" to the already sent message with gpx file(s)

Method 2: Modify gpx2png.py and load local file without the bot
